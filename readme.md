# News App
<p>
	This is a simple News App for android made using kotlin.
	<h1>Features</h1>
	<ul>
		<li>Navigation Drawer</li>
		<li>Intent</li>
		<li>JSON Parsing</li>
		<li>RecyclerView</li>
		<li>Custom ChromeTabs</li>
		<li>Glide</li>
		<li>Volley</li>
	</ul>
</p>
